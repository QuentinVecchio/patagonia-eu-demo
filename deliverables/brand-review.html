DUAL-MISSION TENSION

Commerce and activism compete for primary user intent

The navigation gives equal visual prominence to "Women's / Men's / Kids'" (shopping) and "Activism / Stories / More" (mission). Hero CTAs include both "Explore the Collection" and "Take Action Now" at the same level. The demo needs to decide which thread (product performance OR brand activism) leads the OF1 narrative for Mia Kessler. Splitting attention risk: neither story lands fully.

PROPRIETARY FONT RISK

Ridgeway Sans is a proprietary brand typeface

Patagonia's primary typeface "Ridgeway Sans" is hosted on their own Demandware infrastructure and is not available via Google Fonts or Adobe Fonts. The EDS demo site will need a fallback — Inter or Neue Haas Grotesk are closest visual matches. Copernicus (Galaxie Copernicus) is similarly proprietary. Design should specify explicit system-font fallbacks to avoid FOUT.

SCALE NOTE

Type scale is ad-hoc, not modular

H2 (48px) → H3 (40px) is a 1.2× ratio; H3 → H4 (14px) is a 2.86× drop with no intermediate step. This means prototype typography will need manual size tuning rather than following a predictable scale formula. Acceptable for fidelity reproduction, but note when generating new sections.
